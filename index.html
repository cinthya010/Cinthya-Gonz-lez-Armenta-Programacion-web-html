Etiquetas de contenido

1. Parrafo: <p>

Define un párrafo de texto, es decir, un bloque de contenido que se muestra separado visualmente de otros párrafos.
Creando una pausa natural en la lectura y una distancia predeterminada entre ellos, ideal para estructurar textos en una página web.

2. Pre: <pre> 
Define texto preformateado que mantiene exactamente los espacios, tabulaciones y saltos de línea escritos en el código fuente. 
            
Visualiza el contenido con una fuente monoespaciada (ancho fijo), ideal para mostrar bloques de código, arte ASCII o poesía.
3. Blockquote: <blockquote>
Es una etiqueta HTML semántica para indicar una cita en bloque, es decir, un fragmento de texto extenso tomado de otra fuente, que se muestra visualmente distinto (generalmente con sangría) y puede incluir el atributo cite para la URL de la fuente original.
