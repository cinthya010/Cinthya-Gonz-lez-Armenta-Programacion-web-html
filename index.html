Etiquetas de contenido

1. Párrafo: <p>

Define una parte que debe mostrarse como un párrafo.

2. Pre: <pre> 
Este texto está preformateado.
Conserva espacios y saltos de línea.
Ideal para código o ejemplos.
3. Blockquote: <blockquote>
Representa un contenido citado desde otra fuente.
4. Lista ordenada: <ol>
  1. Luis
  2. Ana
  3. Carlos
  4. Mariana
  5. Antonio
Lista no ordenada: <ul>
  • Perro
  • Gato
  • Pájaro
  • Pez
  • Conejo
5. Lista de términos y definiciones: <dl>
HTML
Lenguaje de Marcas de Hipertexto.
CSS
Hojas de estilo en cascada para diseño visual.
